    Current Focus

    About

    围绕 AI、声音体验与文化叙事，建立一个更有作者感的公开入口。

    Design Principles

    现在这版更强调手绘气质、留白和可读性的平衡。

      Field Notes

      把工作偏好和做事顺序，写成更轻的边栏说明。

        Selected Work

        保留原有内容结构，但把项目区改成更像展陈地图。

        项目不再是标准卡片墙，而是放在一块带线条、边注和纸感的展示面板里。 这样既能保住信息密度，也更接近你想学的小红书参考风格。

        Approach

        从抽象主题到界面语言，方法本身也应该被看见。

        这一版通过手绘边界、慢速线条和更轻的浏览器窗口感，补足了之前页面在区域分布上的失衡， 让内容不再只集中在正中央。

        Now

        接下来最值得补的，是让文字作品集逐步长出真实素材和更完整的项目叙事。

          Contact

          公开入口先保持克制，后续再扩成完整的对外名片。

          目前先让 GitHub 和项目仓库承担主要入口，等你补上真实邮箱、社媒或域名后再继续扩。
