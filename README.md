# GitHub Pages Portfolio Starter

This folder now contains a static personal portfolio homepage that can be deployed directly to GitHub Pages.

## Files

- `index.html`: the new portfolio homepage
- `archive/youth-in-refraction.html`: the previous experimental page, kept as a backup

## Edit your content

Open `index.html` and scroll to the bottom.

Replace the values inside the `portfolioData` object:

- `name`, `role`, `intro`
- `links`
- `stats`
- `projects`
- `practices`
- `now`

You can keep the layout and only change the data.

## Deploy to GitHub Pages

1. Create a GitHub repository.
2. Push this folder to the repository root.
3. On GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your default branch and `/ (root)`.
6. Save and wait for GitHub Pages to publish.

Your site URL will usually be:

- `https://<github-username>.github.io/<repo-name>/`

If you want to publish at `https://<github-username>.github.io/`, use a repository named `<github-username>.github.io`.
