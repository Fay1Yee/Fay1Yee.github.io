Node 1 / Onboarding & Hero

Youth behaves like refracted light.

This single-page exhibition turns refraction, color drift, and shadow bloom into a metaphor for unstable identity. The central material is not a fixed object. It becomes legible only through angle, distance, and interference, mirroring how young selves are read differently under changing social light.

Scroll into the interactive lab, drag the parameters on the right, and let the material slip from lucid to restless. Each state can be captured as a temporary evidence card: an emotional slice rather than a final form.

Node 2 / Interactive Color Lab & Mood Mapping

Parameter logic

Light Angle rotates the illumination vector and reshapes the specular sweep. Interference amplifies prismatic color separation and drives the emotional threshold map. Blur softens contour legibility, making the material feel more fragile, porous, or suspended. All three values are lerped in the render loop so dragging remains smooth and tactile rather than abrupt.

  • Hover states use pearlescent highlights and restore text opacity from muted to fully legible.
  • Mood tags recompute in real time and elevate the strongest descriptor as the active state.
  • Ambient glow follows the cursor with a delayed drift to keep the room feeling spatial.
Node 3 / Material Application Preview

Same light, different surface logic

The shader remains continuous while the CMF layer changes how the refraction reads. Frosted glass blooms and diffuses, ceramic absorbs and grounds, acrylic sharpens and clarifies. The frame styling and fragment shading both shift together, so the preview feels like a material translation rather than a simple skin swap.

The result is deliberately not photoreal 3D. It is a digital material study: a controlled visual metaphor that remains lightweight enough to run as a single HTML file in a desktop browser.

Node 4 / Gallery & Export
